# .vim #

My .vim directory (now with plug)

## Installation ##

    cd ~
    git clone git@github.com:maikrhg/dotvim.git ~/.vim
    ln -s ~/.vim/vimrc ~/.vimrc
    mkdir ~/.vim_tmp (optional)
    cd ~/.vim
    vim

Install Ag (the silver searcher) & ripgrep with Homebrew:

    brew install ag
    brew install ripgrep
